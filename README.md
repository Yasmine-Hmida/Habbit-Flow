# HabbitFlow

**HabbitFlow** is a simple and elegant habit tracker built only with **React**. It allows users to track their daily habits and routines in a visual and organized way.

---

## Features

- Add new habits.
- Delete existing habits.
- Mark habits as complete for each day.
- Visual progress tracking.
- Built using:
  - `useState`
  - `react-router-dom`
  - `props`
  - `useEffect`

## Installation

To run the project locally:

```bash
# Clone the repository:
git clone https://github.com/Yasmine-Hmida/HabbitFlow.git

# Navigate into the project folder:
cd HabbitFlow

# Install Dependencies:
npm install

# Run the App:
npm run dev
